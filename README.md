# FmdlTool
Tool for inspecting .fmdl data.

# Usage
Use cmd and point the tool to a .fmdl or directory.
```
FmdlTool example.fmdl
```

FmdlTool folderpath -OutputHashes

Processes whole directory and outputs hashes and strings.
